# Changelog

## [2.0.0] - 2024-03-24

### Added
- Hub autofill integration — send tagged modules directly to Marfeel Hub
- Coverage ring visualization showing % of page links covered
- Uncovered links highlighting with navigator (prev/next)
- Multi-select mode with Shift+click for pattern detection
- DOM tree navigation with Alt+Scroll during picking
- Breadcrumb trail showing ancestor chain
- Module detection for existing Marfeel `data-mrf-recirculation` attributes
- 8-color overlay palette for tagged modules
- Human-readable module naming from element attributes

### Changed
- Upgraded to Chrome Manifest v3
- Side panel UI replaces popup

## [1.0.0] - 2024-03-22

### Added
- Initial release
- Visual element picker
- CSS selector generation
- Basic overlay system
- Python analyzer (`analyze_recirculation.py`) with Playwright
